This plugin provides a JSON RPC API for WordPress.

== Installation ==

1. Upload  the `wp-json-rpc-api` directory to your `/wp-content/plugins/` directory, or include `wp-json-rpc-api/wp-json-rpc-api.php` in one of your own plugins.
1. Activate it through the 'Plugins' menu in WordPress
