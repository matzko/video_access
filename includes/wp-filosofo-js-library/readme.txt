== Description ==
A library of common JS functionality used by Austin Matzko (filosofo) in WordPress plugins.

== Installation ==

1. Upload the `wp-filosofo-js-library` directory to the `/wp-content/plugins/` directory, or include `wp-filosofo-js-library/wp-filosofo-js-library.php` in your own plugin.
1. Activate the plugin through the 'Plugins' menu in WordPress
